Parameters
Name Description Details
api_key

The API Key.

 string, optional
example: myApiKey
row_limit

The maximum number of rows to return.

 integer, optional
example: 5000
default: 1000
constraint: 1 <= row_limit <= 33000
primary_currency

The code of the currency to return sales data back in in the case where there are conflicts (multiple currencies) in the dataset.

string, optional
example: EUR
fallback_currency

The code of the currency to return sales data back in in the case where there are conflicts (multiple currencies) in the dataset and no primary currency is specified or conversion to the primary currency failed (for example, when exchange rates are unknown).

string, optional
example: EUR
default: USD
calendar_type

The calendar type to use in queries.

 enum, optional
example: retail
possible values:
gregorian retail
default: gregorian
location_ids

Filter by specific location ids.

Note that corp is an alias for the id of the hosting server.

 string, optional
example:
  66b64d81-73da-4aca-83e8-cf14740a64b4
default: corp
measures

The measures to query.

string, required
example: sales;traffic+subtype=out
groupings

Specifies how to group the output.

 string, required
example: location;year_month+verbose=true
date_ranges

Filter with date ranges.

 string, optional
example: y2014
default: previous2weeks
constraint: date_ranges >= 2009
time_ranges

Filter with time ranges.

string, optional
example: 0630-1800;store_hours
