Groupings
Attributes
Name Description Details
alias

Names the corresponding grouping field in the output JSON.

Field names match their corresponding id by default, however, an alias must be specified if an id is used more than once since all output fields must be unique.

 string, optional
example: loc_id
verbose

For time-based groupings, the fields range_start, range_end and is_complete are added to the JSON output.

 boolean, optional
example: true
