@(hostaddress: String) @import java.net.URLEncoder.encode Creating PNG images from LaTeX with minimal URLs

TexUp

Easy LaTeX to PNG without embedding Javascript

Given LaTeX:

       => \frac{a}{b}

URL-encode the LaTeX with this site

       => %5Cfrac%7Ba%7D%7Bb%7D

Access the LaTex PNG at:

       => @{hostaddress + "/" + encode("\\frac{a}{b}", "UTF-8")}
